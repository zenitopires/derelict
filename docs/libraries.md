# Libraries
For this project, I'm using several libraries to facilitate things like logging, math, string formatting. Here's an exhaustive list of said libraries.

# Third Party Libraries Used

| Library                                                                                 | Notes                                                                                                   |
|-----------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------|
| [SimpleDirect Media Layer (SDL)](https://github.com/libsdl-org/SDL)                     | SDL facilitates the window that OpenGL can use to render objects. [More Info](https://www.libsdl.org/)  |
| [OpenGL Mathematics (GLM) ](https://github.com/g-truc/glm)                              | Math library to do all the vector and matrices math.                                                    |
| [fmt](https://github.com/fmtlib/fmt)                                                    | C++ string formatting library.                                                                          |
| [glad](https://github.com/Dav1dde/glad)                                                 | glad is used to load the system's OpenGL function pointers to easy to use functions in the application. |
| [spdlog](https://github.com/gabime/spdlog)                                            | C++ logging library.                                                                                    |
